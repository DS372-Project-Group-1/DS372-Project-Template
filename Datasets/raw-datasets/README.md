This folder has all the datasets we got from different sources for the full integration:
1. Wildfires data: from Ontario GeoHub: https://geohub.lio.gov.on.ca/datasets/lio::fire-disturbance-point/about
2. Weather data: we got the data for each target city from Open-Meteo: https://open-meteo.com
3. PM2.5 data (Air Pollutant): Google Engine
